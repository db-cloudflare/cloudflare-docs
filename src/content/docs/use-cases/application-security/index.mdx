Protect your website or application from attacks, bots, and abuse with Cloudflare's security solutions.

Whether you have an existing website hosted elsewhere or are building something new, Cloudflare sits in front of your application to block threats before they reach your origin server. This guide walks you through the security solutions available to protect your web applications.

---

## Encrypt traffic with SSL/TLS

Secure all connections between visitors and your application with automatic HTTPS.

### Solutions

<CardGrid>

<LinkTitleCard title="SSL/TLS" href="/ssl/" icon="ssl">

</LinkTitleCard>

<LinkTitleCard title="SSL for SaaS" href="/cloudflare-for-platforms/cloudflare-for-saas/security/certificate-management/" icon="ssl">

Provision SSL certificates for custom domains at scale.

</LinkTitleCard>

</CardGrid>

### What you get

- **Free certificates** - Universal SSL certificates at no cost
- **Automatic renewal** - Certificates renew automatically before expiration

## Block application attacks

Protect against SQL injection, XSS, and other OWASP Top 10 vulnerabilities.

### Solutions

<CardGrid>

<LinkTitleCard title="Application Security (WAF)" href="/waf/" icon="waf">

Managed rulesets that block common attack patterns automatically.

</LinkTitleCard>

<LinkTitleCard title="Custom Rules" href="/waf/custom-rules/" icon="waf">

Create custom firewall rules based on your application's needs.

</LinkTitleCard>

<LinkTitleCard title="Rate Limiting" href="/waf/rate-limiting-rules/" icon="waf">

Protect endpoints from abuse with request rate limits.

</LinkTitleCard>

</CardGrid>

### What you get

- **Managed rulesets** - Pre-configured rules for OWASP Top 10 and emerging threats
- **Zero-day protection** - Cloudflare updates rules as new vulnerabilities are discovered
- **Custom rules** - Block or challenge requests based on any request attribute
- **Rate limiting** - Prevent brute force and API abuse attacks

## Stop malicious bots

Detect and block automated threats while allowing legitimate traffic.

### Solutions

<CardGrid>

<LinkTitleCard title="Bot Security" href="/bots/" icon="bots">

ML-powered bot detection with granular control over bot traffic.

</LinkTitleCard>

<LinkTitleCard title="Super Bot Fight Mode" href="/bots/get-started/pro/" icon="bots">

Basic bot protection included with Pro plans and above.

</LinkTitleCard>

<LinkTitleCard title="Turnstile" href="/turnstile/" icon="turnstile">

Privacy-preserving CAPTCHA alternative for forms and sensitive pages.

</LinkTitleCard>

</CardGrid>

### What you get

- **Bot scores** - Every request receives a bot score from 1 (bot) to 99 (human)
- **Verified bots** - Allow known good bots like search engine crawlers
- **Challenge pages** - Present challenges to suspicious traffic
- **Form protection** - Protect login, signup, and checkout forms

### Get started

1. [Enable Bot Fight Mode](/bots/get-started/free/) (Free plan)
2. [Configure Super Bot Fight Mode](/bots/get-started/pro/) (Pro plan and above)
3. [Add Turnstile to forms](/turnstile/get-started/)

## Secure API endpoints

Protect APIs with schema validation, rate limiting, and authentication.

### Solutions

<CardGrid>

<LinkTitleCard title="API Shield" href="/api-shield/" icon="api-shield">

Discover, secure, and monitor your APIs.

</LinkTitleCard>

<LinkTitleCard title="mTLS" href="/ssl/client-certificates/" icon="ssl">

Authenticate API clients with mutual TLS certificates.

</LinkTitleCard>

</CardGrid>

### What you get

- **API discovery** - Automatically find API endpoints in your traffic
- **Schema validation** - Reject requests that do not match your OpenAPI spec
- **Sequence mitigation** - Detect and block API abuse patterns
- **mTLS authentication** - Require client certificates for API access

For the complete guide, refer to [API Shield get started](/api-shield/get-started/).
