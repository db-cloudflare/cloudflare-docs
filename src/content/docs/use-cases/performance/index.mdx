Accelerate your website or application with global caching, image optimization, and smart routing.

Whether you have an existing website or are building something new, Cloudflare's global network delivers your content faster by caching it closer to your visitors. This guide walks you through the performance solutions available to speed up your web applications.

---

## Cache content globally

Serve static and dynamic content from Cloudflare's edge network, reducing latency and origin load.

### Solutions

<CardGrid>

<LinkTitleCard title="Cache" href="/cache/" icon="cache">

Cache static assets and HTML at 300+ edge locations worldwide.

</LinkTitleCard>

<LinkTitleCard title="Cache Rules" href="/cache/how-to/cache-rules/" icon="cache">

Fine-grained control over what gets cached and for how long.

</LinkTitleCard>

<LinkTitleCard title="Tiered Cache" href="/cache/how-to/tiered-cache/" icon="cache">

Reduce origin requests by using regional cache tiers.

</LinkTitleCard>

<LinkTitleCard title="Cache Reserve" href="/cache/advanced-configuration/cache-reserve/" icon="cache">

Persistent cache storage for long-tail content.

</LinkTitleCard>

</CardGrid>

### What you get

- **Global distribution** - Content cached in 300+ locations worldwide
- **Reduced latency** - Visitors get content from the nearest edge location
- **Origin offload** - Fewer requests reach your origin server
- **Bandwidth savings** - Serve cached content without origin bandwidth costs

### Configure caching

<Details header="Set up cache rules">

1. Go to the [Cloudflare dashboard](https://dash.cloudflare.com/) and select your zone.
2. Go to **Caching** > **Cache Rules**.
3. Select **Create rule**.
4. Configure caching behavior:
   - **Static assets**: Cache with long TTLs (e.g., 1 year for versioned assets)
   - **HTML pages**: Cache with shorter TTLs or use `stale-while-revalidate`
   - **API responses**: Cache selectively based on endpoint

Example cache rule for static assets:
- **Expression**: `http.request.uri.path.extension in {"js" "css" "png" "jpg" "gif" "svg" "woff2"}`
- **Cache eligibility**: Eligible for cache
- **Edge TTL**: 1 year

</Details>

## Optimize images

Automatically compress, resize, and convert images to modern formats.

### Solutions

<CardGrid>

<LinkTitleCard title="Images" href="/images/" icon="images">

Transform, optimize, and deliver images at scale.

</LinkTitleCard>

<LinkTitleCard title="Polish" href="/images/polish/" icon="images">

Automatic image compression without quality loss.

</LinkTitleCard>

</CardGrid>

### What you get

- **Automatic compression** - Reduce image file sizes without visible quality loss
- **Modern formats** - Serve WebP and AVIF to supported browsers
- **Responsive images** - Resize images on-the-fly for different devices
- **Lazy loading** - Defer loading of off-screen images

### Configure image optimization

<Details header="Enable Polish and WebP">

1. Go to the [Cloudflare dashboard](https://dash.cloudflare.com/) and select your zone.
2. Go to **Speed** > **Optimization** > **Image Optimization**.
3. Turn on **Polish** and select compression level:
   - **Lossless**: No quality loss, smaller file size reduction
   - **Lossy**: Slight quality reduction, larger file size reduction
4. Turn on **WebP** to serve modern formats to supported browsers.

</Details>

For the complete guide, refer to [Images documentation](/images/).

## Optimize web assets

Automatically minify and optimize HTML, CSS, and JavaScript.

### Solutions

<CardGrid>

<LinkTitleCard title="Speed" href="/speed/" icon="speed">

Automatic optimizations for web assets and Core Web Vitals.

</LinkTitleCard>

<LinkTitleCard title="Zaraz" href="/zaraz/" icon="zaraz">

Load third-party tools without blocking page rendering.

</LinkTitleCard>

</CardGrid>

### What you get

- **Minification** - Remove unnecessary characters from HTML, CSS, and JavaScript
- **Compression** - Brotli and Gzip compression for all text assets
- **Third-party optimization** - Load analytics and marketing tags efficiently
- **Core Web Vitals** - Improve LCP, FID/INP, and CLS scores

### Configure optimizations

<Details header="Enable Speed optimizations">

1. Go to the [Cloudflare dashboard](https://dash.cloudflare.com/) and select your zone.
2. Go to **Speed** > **Optimization**.
3. Enable optimizations:
   - **Auto Minify**: Minify HTML, CSS, and JavaScript
   - **Brotli**: Enable Brotli compression
   - **Rocket Loader**: Defer JavaScript loading
   - **Mirage**: Optimize image loading on mobile

</Details>

For the complete guide, refer to [Speed documentation](/speed/).
